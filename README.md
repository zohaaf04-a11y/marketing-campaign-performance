# Marketing Campaign Performance Dashboard

##  Project Overview

This project focuses on analyzing marketing campaign performance using Microsoft Excel. The goal was to evaluate campaign effectiveness by analyzing key performance metrics and presenting actionable insights through an interactive dashboard.

---

##  Objective

The objective of this project was to:

* Analyze marketing campaign data.
* Identify trends in engagement and conversions.
* Evaluate campaign performance across different channels and audience segments.
* Create an interactive dashboard to support data-driven decision-making.

---

##  Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Dashboard Design
* Microsoft Word (Report Documentation)

##  Project Workflow

The following steps were performed during the analysis:

1. Loaded and explored the marketing campaign dataset.
2. Performed data cleaning and validation.
3. Analyzed key performance metrics such as:

   * Engagement
   * Clicks
   * Conversions
   * Campaign performance
4. Created Pivot Tables to summarize the data.
5. Built Pivot Charts for data visualization.
6. Designed an interactive dashboard using Excel.
7. Interpreted findings to generate business insights.

---

##  Dashboard Features

* Campaign performance overview
* Engagement analysis
* Conversion analysis
* Channel-wise performance
* Interactive Pivot Charts
* Clear and easy-to-understand visualizations

---

##  Key Insights

* Campaign performance varies across different marketing channels.
* Higher engagement generally leads to improved conversion rates.
* Some campaign types perform significantly better within specific audience segments.
* Interactive dashboards enable faster comparison of campaign performance and support informed business decisions.
