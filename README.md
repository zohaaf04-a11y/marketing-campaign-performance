

Marketing Campaign Analysis & Dashboard

Objective
The objective of this task was to analyze marketing campaign performance using Excel-based data analysis and create a dashboard to visualize key insights such as engagement, conversions, and overall campaign effectiveness.

Tools Used
- Microsoft Excel
- Data Visualization (Charts & Pivot Tables)
- Dashboard Design Techniques
- MS Word (Report Documentation)

Dataset
Marketing Campaign Dataset

Files Included
- marketing_campaign_dataset.xlsx → Raw Dataset
- Marketing Campaign EDA Report.docx → Exploratory Data Analysis Report
- marketing campaign performance dashboard.png → Final Dashboard Visualization
- README.md → Project Documentation

Data Analysis & Dashboard Steps
- Loaded and explored the marketing campaign dataset
- Performed data cleaning and basic validation
- Analyzed key metrics such as engagement and conversion trends
- Created pivot tables for summarizing data
- Designed a dashboard using charts and visual elements
- Interpreted insights for business decision-making

Key Insights
- Campaign performance varies across different channels and target audiences
- Engagement levels strongly influence conversion rates
- Certain campaign types perform better in specific segments
- Visual dashboards help in quick comparison and decision-making
